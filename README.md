# AI Document Summarizer

This project is an AI-powered document summarization system using Azure OpenAI.

## Features
- Upload PDF documents
- Generate summaries using LLM
- Extract key points
- Context-aware summarization (RAG)

## Tech Stack
- Azure OpenAI
- Python / .NET
- FAISS / Azure AI Search

## How it works
1. Upload document
2. Split into chunks
3. Generate embeddings
4. Retrieve relevant data
5. Generate summary
